# usage

```sh
npm i react-text-settle
```

```js
import titleCondition from "react-text-settle";

<div className="title" {...titleCondition}>
  very very long text
</div>;
```

# applicable

- `ellipsis` & `title tag`

```css
 {
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}
```
