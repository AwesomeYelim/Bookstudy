
# **typescript와 함께 사용하는 Redux toolkit의 내장함수**

**createSlice 를 사용해보자 !**

📌 createSlice 란?

    reducer 뿐만 아닌 action을 포괄하는 redux toolkit 의 내장함수이다.
    action 타입을 inline 요소로 제공함으로써 기존의 코드를 더욱 가독성 있는코드로 정리해준다.
